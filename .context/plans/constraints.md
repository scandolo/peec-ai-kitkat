# Hackathon Constraints

## Deadlines
- **Sunday 14:00** — project submission deadline (competition opt-in)
- **Sunday 15:00** — finalists announced
- **Sunday 15:15** — finalist pitches (5 min each)

## Submission requirements
- Team of max 5 people
- Use min 3 partner technologies (Google Deepmind, Lovable, Gradium, Entire, Tavily, Pioneer — Aikido doesn't count)
- Project must be created newly at the hackathon (boilerplates allowed)
- 2-minute video demo (Loom or equivalent)
- Public GitHub repo with README, setup instructions, API/tool documentation

## Competition format
- **Stage 1**: 8 finalist teams selected (1 per track) — judged on creativity, technical complexity, partner tech usage
- **Stage 2**: 5-min live pitch → jury picks top 3
