# kitkat-bulls-GEO

Berlin Hackathon 2026 - Team KitKat Bulls

## Track: Peec AI - 0 -> 1 AI Marketer

Building an AI-powered tool that helps early-stage brands win distribution against bigger competitors in AI search. Uses the Peec AI MCP to surface visibility gaps across AI engines (ChatGPT, Perplexity, Gemini, Claude, etc.) and generate actionable strategies.

**Example brands** (test projects provided): Nothing Phone vs Apple/Samsung, Attio vs Salesforce/HubSpot, BYD vs Tesla.

**Docs**: https://docs.peec.ai/mcp/introduction
